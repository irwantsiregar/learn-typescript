Kenapa harus memilih TypeScript?
- Trends : Berdasarkan survey
- Code Quality / Kualitas Kode : Meningkatkan code readability/kemudahan untuk membaca codebase, khususnya oleh rekan tim atau developer lainnya.
- Mengurangi bugs: Dapat meminimalisir terjadinya bugs. Berdasarkan penelitian TypeScript dapat mendeteksi bugs sebanyak 15%. Angka yang cukup besar bagi developer untuk dapat mendeteksi bugs sejak dini (deveplopment process).
- Scalable : Kemudahan bagi developer untuk dapat melakukan Refactor code/ perubahan requirement/ penambahan feature.
